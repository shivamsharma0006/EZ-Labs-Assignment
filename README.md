# EZ Labs Contact Form Assignment

## Overview
This project implements a contact form built using **React + Tailwind CSS**.  
It connects with the provided backend API to send form data and handle validation using the `validator` library.

---

## Features
- Beautiful UI inspired by the provided Figma design
- Form validation (name, email, phone, message)
- Integration with the given API endpoint
- Real-time error handling
- Tested API using **Postman**
